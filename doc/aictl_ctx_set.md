## aictl ctx set

Set current aictl configuration

```
aictl ctx set [flags]
```

### Options

```
  -b, --branch-id string    Branch id
  -h, --help                help for set
      --no-tls-skip         Not skip certificate verification
  -p, --project-id string   Project id
      --tls-skip            Skip certificate verification
  -t, --token string        AI server access token
  -u, --uri string          AI server uri
```
